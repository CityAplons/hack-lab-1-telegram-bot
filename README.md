## This project was created for hackaton.

### A telegram bot based capital game.

Requirements: NodeJS 12

Libraries: 
  * telegraf, 
  * telegraf-inline-menu, 
  * telegraf-session-local

Install:
```
    npm install
```
Run:
```
    npm start
```
